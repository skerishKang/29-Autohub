# 🚀 AutoHub - 스마트폰 자동화 중계 플랫폼

**분리 프로그램 방식의 스마트폰 문자/알림 자동화 서비스**

> "모든 스마트폰 통신을 하나로 연결하여 AI 기반 자동화를 구현하는 완벽한 솔루션"

---

## 📋 프로젝트 개요

AutoHub는 안드로이드 스마트폰의 **문자(SMS)와 앱 알림**을 자동으로 수집하여 외부 서비스(n8n, 텔레그램 등)로 중계하는 자동화 플랫폼입니다. 개인용 MVP를 통한 안정성 검증 후, **구독/크레딧 기반의 유료 SaaS 서비스**로 확장하는 것을 목표로 합니다.

### 🎯 핵심 가치 제안
- **완벽한 자동화**: 스마트폰 모든 통신을 놓치지 않고 수집
- **AI 기반 필터링**: 스팸 구분, 중요 메시지 추출, 감성 분석
- **유연한 중계**: n8n, 텔레그램, 웹훅 등 다양한 외부 서비스 연동
- **개인정보 보호**: 사용자가 직접 선택하는 민감정보 마스킹 기능

## 📱 서비스 구성

### 🔤 SMSAutoHub (문자 자동 중계)
- **기능**: 모든 수신 문자를 실시간으로 외부 서비스로 전송
- **필터링**: 발신자/키워드 기반 선택적 중계
- **마스킹**: 민감정보(계좌, 인증번호 등) 자동 마스킹 옵션
- **중계 대상**: n8n 웹훅, 텔레그램 봇, 자체 API

### 🔔 NotifyAutoHub (알림 자동 중계)
- **기능**: 모든 앱 알림을 실시간으로 수집 및 중계
- **선택적 수집**: 사용자가 지정한 특정 앱만 필터링 가능
- **알림 종류**: 카카오톡, 이메일, 배송, 금융, SNS 등 모든 앱
- **요약 전송**: 알림 제목/요약/내용 일부 전달 (원문/마스킹 선택)

## 🛠️ 기술 스택

### 📱 모바일 (안드로이드)
- **개발 언어**: Kotlin / Java
- **아키텍처**: MVVM + Clean Architecture
- **권한 관리**: SMS_READ, POSTED_NOTIFICATIONS, INTERNET
- **데이터 저장**: Room DB + SharedPreferences

### 🖥️ 백엔드
- **서버**: Node.js (Express) / Python (FastAPI)
- **데이터베이스**: PostgreSQL / MongoDB
- **메시지 큐**: Redis Pub/Sub
- **웹훅**: n8n 연동 지원

### 🤖 AI 자동화
- **필터링**: OpenAI GPT-4 / Claude API
- **분류**: 스팸/중요/긴급/감성 분석
- **요약**: 메시지 자동 요약 및 태깅
- **자동화**: n8n 워크플로우 연동

### 📱 외부 연동
- **텔레그램**: Bot API를 통한 실시간 알림
- **n8n**: 웹훅을 통한 자동화 워크플로우
- **Slack**: Webhook을 통한 팀 알림
- **Email**: SMTP를 통한 이메일 알림
